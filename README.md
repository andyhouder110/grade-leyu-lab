# grade-leyu-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面作为静态资源存放，不针对任何特定域名或具体网站，仅供查阅与参考。

## 项目简介

grade-leyu-lab 是一个轻量级的页面归档仓库，主要用于收集、整理并展示各类独立 HTML 文件。仓库本身不提供动态服务或后端功能，所有页面均为纯前端静态内容。

## 目录说明

仓库根目录下包含以下内容：

- `index.html` — 主入口页面，用于导航或索引
- `pages/` — 存放各个独立 HTML 页面的文件夹
- `assets/` — 存放页面所需的样式、脚本、图片等资源文件

每个 HTML 页面均独立存在，不依赖其他页面或外部服务。

## 页面归档说明

所有归档页面均为完整的独立 HTML 文件，可直接在浏览器中打开查看。归档内容可能包括但不限于：

- 信息展示页
- 工具或演示页面
- 实验性前端页面

归档页面不定期更新，具体内容以仓库内文件为准。

## 维护说明

- 本仓库由维护者不定期整理与更新
- 不保证所有页面在任意浏览器或设备上完全兼容
- 欢迎通过 Issues 提交建议或报告问题
- 不接受针对特定网站或商业用途的请求

## 使用方式

1. 克隆或下载本仓库
2. 直接在本地浏览器打开 `index.html` 或 `pages/` 下的任意 HTML 文件
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问
